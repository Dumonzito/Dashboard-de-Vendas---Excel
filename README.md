# Dashboard-de-Vendas---Excel
Dashboard de vendas, com foco na organização e visualização de dados, baseado em vendas de mensalidades do XBOX
O projeto foi realizado de forma assíncrona com as aulas do Desafio de Projeto.
Foram utilizados os comandos/recursos de Tabela Dinâmica, Segmentação de Dados, além de configurações personalizadas.
